# Audios Cardíacos

Esta carpeta contiene los archivos de audio de sonidos cardíacos.

## Archivos esperados:

- normal_heart.mp3
- soplo_estenosis_aortica.mp3
- soplo_insuficiencia_mitral.mp3
- clic_estenosis_mitral.mp3
- galope_s3.mp3
- roce_pericardico.mp3

## Fuentes recomendadas para descargar audios:

1. https://www.thinklabs.com/heart-sounds/
2. https://www.easyauscultation.com/
3. https://www.heartrhythmjournal.com/
4. https://www.med.umich.edu/lrc/psb_open/html/repo/primer_heartsound/primer_heartsound.html

## Instrucciones:

1. Descarga los audios MP3 de las fuentes
2. Renómbralos según la lista anterior
3. Colócalos en esta carpeta
